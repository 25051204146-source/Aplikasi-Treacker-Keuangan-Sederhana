# 💰 Expense Tracker App

A responsive web-based expense tracker application built using **HTML, CSS, and Vanilla JavaScript**. This project allows users to record, edit, search, categorize, and manage their financial transactions with automatic balance calculation and persistent local storage.

---

## 📖 About

Expense Tracker App helps users manage their personal finances by recording income and expense transactions. The application automatically updates the current balance and stores all transaction data in the browser using **Local Storage**, allowing the data to persist even after the page is refreshed.

---

## ✨ Features

- ➕ Add new transactions
- ✏️ Edit existing transactions
- 🗑 Delete transactions
- 🔄 Change transaction type (Income ↔ Expense)
- 🔍 Search transactions by title
- 📊 Automatic balance calculation
- 💾 Persistent data using Local Storage
- 📱 Responsive design for desktop and mobile
